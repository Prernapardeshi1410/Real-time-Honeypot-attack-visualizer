# Real-time-Honeypot-attack-visualizer
--> Objective
Demonstrate how honeypots work in cybersecurity
Simulate real-world attack scenarios
Provide hands-on understanding of attacker behavior
Build a foundation for advanced intrusion detection systems
Showcase real-time monitoring and visualization techniques

> Technologies Used
Python 3

Socket Programming

Flask (Web Framework)

HTML / CSS

JSON (Log Storage)

Windows Batch Script (One-click execution)

--> How to Run the Project

Step 1: Install Python

Make sure Python 3 is installed. Check using:

python --version

Step 2: Install Required Library pip install flask

Step 3: Run the Project (One Click)

Just double-click:

start_project.bat

This will:

Start the honeypot

Simulate attacks

Open the live dashboard

Step 4: View Live Logs

Open browser:

http://127.0.0.1:5000

You will see:

Timestamp

IP address

Attack type

Severity

Username & password used

--> Main Features

Fake login honeypot

Randomized attack simulation

Severity classification (INFO, WARNING, CRITICAL)

JSON-based structured logging

Live web dashboard
