# Real-time-Honeypot-attack-visualizer
A next-generation cyber deception honeypot system with real-time attack monitoring, intelligent threat classification, and an interactive security dashboard. This project simulates realistic cyber attacks, detects brute-force and credential-based threats, and visualizes attacker behavior through live analytics.
Real-time attack detection & logging
Threat classification (Critical / Warning / Info)
Brute force & credential attack simulation
Live dashboard with analytics (charts, activity tracking)
IP tracking & attacker profiling
Auto-refresh monitoring system
Interactive UI (SOC-style dashboard)
